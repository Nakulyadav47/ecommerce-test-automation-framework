# 🛒 E-Commerce QA Automation Testing Project

## 📌 Overview
This project demonstrates end-to-end QA Automation testing of an e-commerce web application using Selenium WebDriver with Python. It covers complete user flow from login to checkout.

---

## 🚀 Features
- Automated Login Functionality
- Product Selection & Add to Cart
- Cart Validation
- Checkout Process Automation
- End-to-End User Flow Testing

---

## 🛠 Tech Stack
- Python
- Selenium WebDriver
- ChromeDriver

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/yourusername/your-repo-name.git

2. Navigate to project folder:
   cd your-repo-name

3. Create virtual environment:
   python -m venv venv

4. Activate virtual environment:
   venv\Scripts\activate

5. Install dependencies:
   pip install selenium

---

## ▶️ How to Run Tests

Run individual test files:

python tests/test_login.py  
python tests/test_cart.py  
python tests/test_checkout.py
